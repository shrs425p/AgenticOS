  <trust_boundaries>
    - Host operating system registry configuration boundary.
    - Application error handling boundary (catching and formatting AgentError).
  </trust_boundaries>
  <stride_register>
    - **Spoofing**: N/A for this plan.
    - **Tampering (T-01-04 / Registry Tampering)**: Attackers modify system keys (Run, RunOnce, Services) to establish persistent backdoors or alter system behaviors. Resolved by cfg-driven registry key blocking/prompting.
    - **Repudiation**: N/A for this plan.
    - **Information Disclosure**: Unauthorized access to system keys. Resolved by registry policies.
    - **Denial of Service**: Corrupting system registry hives. Mitigated by HKLM/Software/System blocks.
    - **Elevation of Privilege**: Altering run/startup keys to run arbitrary code on boot. Mitigation intercepts and blocks these commands.
  </stride_register>
</threat_model>
