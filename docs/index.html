Quick-Start Installation

Get AgenticOS running in your local sandbox environment in under two minutes.

# 1. Clone & Install dependencies pip install -r requirements.txt playwright install chromium # 2. Run Setup to link the 'agent' globally .\setup.ps1 # 3. Spin up the Control Console agent
  • Requires Python 3.10 or higher.
  • Runs fully client-side and portable on standard command terminals.

Cortex Reasoning Loop

AgenticOS runs on a structured, self-correcting Think-Action-Observation cycle, preventing cascading pipeline errors.

  • Cognitive Orchestrator: Manages multi-modal context buffers and prompt templates.
  • Core Memory Registry: Leverages a local SQLite database storing task histories, tool results, and learned success heuristics.
  • API Dispatch Layer: Manages connections to Ollama and Cloud Models with resilient 429 exponential backoffs.

Zone-Based PathGuard

AgenticOS segregates the filesystem into strict security zones, protecting operational integrity.

  • Green Zone (workspace/): Full autonomous access for research digests, reports, and code testing.
  • Yellow Zone (System Folders): Human-in-the-Middle (HITM) triggers require active console approvals for modifications.
  • Red Zone (Kernel/Registry): Blocked automatically at the tool registry layer to prevent dangerous alterations.

Common Resolution Pathways

Easy troubleshooting tips for system integration blocks.

  • Windows COM Audio Failures: Run Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process to authorize C# COM compiler loads.
  • macOS accessibility blockages: Enable accessibility privileges for your terminal emulator under System Settings -> Privacy -> Accessibility.
  • Rate limits: Built-in retry_call() backoff handles 429 errors autonomously.
